# NavBar
You have too many items to be displayed in the menu.
You want to show the links vertically.
### NavBars são componentes utilizadas para criar soluções de navegação e menus.
---
<br />

## Recomendamos o uso de NavBars quando:
- Você precisa separar conteúdo por páginas.
- Não for apropriado dividir conteúdo usando tabs, sidebars ou usando cards e links para navegar entre páginas.

<br />

## Não recomendamos o uso de NavBars quando:
- V.
- Você quiser construir componentes de navBaregação (Menu, navBarbar, etc).
- As ações listadas afetam toda a view ou mais que um conjunto de dados relacionados.

<br />
