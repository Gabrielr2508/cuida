# StatusList

### Actions lists são componelistntes usados para mostrar uma lista de ações em linha, podendo algumas delas serem ocultas.
---

#### Recomendamos o uso de StatusLists quando:
- Você possuir ações associadas a um item de uma lista ou linha de tabela.
- Houverem várias ações possíveis em determinada situação, mas algumas delas puderem ser colapsadas.

#### Não recomendamos o uso de StatusLists quando:
- Você estiver procurando por botões.
- Você quiser construir componentes de navegação (Menu, navbar, etc).
- As ações listadas afetam toda a view ou mais que um conjunto de dados relacionados.
